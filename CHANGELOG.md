## 1.0.0

* Initial release of `cv_builder`.
* Introduced `CVModel` for data management and JSON serialization.
* Added `FormScreen` for interactive CV building with support for:
    * Basic Information
    * Dynamic Education entries
    * Dynamic Experience entries
    * Skill levels (Beginner to Expert)
* Added `PreviewScreen` for a clean, modern CV layout with JSON export functionality.
* Included `SectionCard` widget for UI consistency.
